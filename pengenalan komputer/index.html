Apa Itu Komputer?

Komputer adalah perangkat elektronik yang digunakan untuk mengolah data menjadi informasi. Komputer dapat menerima input, memproses data, menyimpan data, dan menghasilkan output.

Komponen Utama Komputer

1. Hardware

Perangkat keras seperti monitor, keyboard, mouse, CPU, dan printer.

2. Software

Perangkat lunak seperti sistem operasi (Windows, Linux) dan aplikasi (Word, Excel).

3. Brainware

Pengguna komputer seperti pelajar, guru, dan teknisi yang mengoperasikan perangkat.

Fungsi Komputer

No Fungsi Penjelasan
1 Input Menerima data dari pengguna melalui perangkat seperti keyboard dan mouse.
2 Proses Mengolah data menggunakan prosesor (CPU) sesuai instruksi program.
3 Output Menampilkan hasil proses melalui monitor atau printer.
4 Storage Menyimpan data dalam media seperti harddisk, SSD, atau flashdisk.
